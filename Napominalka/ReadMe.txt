Небольшая напоминалочка на C# для напоминания дня рождения все в той же войсковой части, где запрещено использовать смартфоны, 
отсутствует интернет, а забывать про дни рождения не совсем красиво. Программа для работы на windows server.
Файл People - основной класс, для отображения ФИО и даты рождения;
Файл MainWindow - реализует сверку с системными часами сервера и если день рождения "на подходе" трубит, чтоб не забывали.

Сама задумка не глобальная, поэтому я решил, в данном случае, не делить View от Modal.
