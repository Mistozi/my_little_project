Миничат я писал для бухгалтерского отдела войсковой части, где запрещается использование месенджеров и отсутствует доступ в интернет.
Файл TCP - организует работу по протоколу TCP/IP, такую как: проверку порта, передача текстовой информации.
Файл MainWindow - организует работу с пользователем.
Файл Messager - класс, для сохранения и отображения сообщения.
